# 项目需求说明书 (Requirements Specification)

## 1. 项目概述
本系统旨在为室内装饰施工图部门提供一个轻量级、可视化的项目管理平台。通过数字化手段替代传统的 Excel/纸质记录，实现项目进度的实时透明化，方便主管监控资源分配及项目状态。

## 2. 用户角色
- **普通员工 (Employee)**:
  - 查看自己负责的项目列表。
  - 更新自己负责项目的进度状态、实际使用时间等字段。
- **部门主管 (Admin/Manager)**:
  - 查看所有项目（进行中、历史、延期等）。
  - 创建新项目，指派负责人。
  - 编辑任意项目信息。
  - 导出数据报表。

## 3. 功能模块详情

### 3.1 首页仪表盘 (Dashboard)
- **关键指标卡片**:
  - 当前进行中项目总数。
  - 本月即将到期项目数。
  - 延期项目警示数。
- **项目列表视图**:
  - 默认展示状态为“进行中”的项目。
  - 支持按“项目名称”、“负责人”、“状态”进行筛选。
  - 列表字段：项目编号、项目名称、负责人、预计结束时间、当前状态、操作（查看详情）。

### 3.2 员工视图 (Employee Profile)
- **入口**: 点击首页列表中的负责人姓名，或侧边栏选择特定员工。
- **内容**:
  - 员工基本信息（姓名、职位）。
  - **正在进行的项目**: 列表展示。
  - **历史已完成项目**: 列表展示（可折叠或分页）。
  - 统计图表：该员工近半年的项目完成数量趋势（可选二期实现）。

### 3.3 项目详情与管理 (Project Management)
- **新建/编辑项目表单**:
  - **基础信息**:
    - 项目编号 (String, 唯一, 自动生成或手动输入)
    - 项目名称 (String, 必填)
    - 负责人 (Select, 关联员工表)
  - **时间管理**:
    - 开始时间 (Date)
    - 预计结束时间 (Date)
    - 实际结束时间 (Date, 仅当状态为“已完成”时必填/显示)
  - **甲方信息**:
    - 对接人姓名 (String)
    - 联系电话 (String)
    - 邮箱 (String, Email格式校验)
  - **状态控制**:
    - 当前状态 (Enum: 未开始, 进行中, 待审核, 已完成, 延期)
    - 备注/描述 (Text)
- **权限控制**:
  - 只有授权用户（主管或项目负责人）可见“编辑”按钮。

### 3.4 数据导出 (Export)
- **功能**: 在首页或项目列表页提供“导出 Excel”按钮。
- **逻辑**:
  - 根据当前页面的筛选条件（如：只导出“延期”项目）生成数据。
  - 若无筛选，默认导出所有项目。
  - 文件格式: `.xlsx` (兼容 WPS/Office Excel)。
  - 包含字段：项目编号, 名称, 负责人, 开始时间, 预计结束, 实际结束, 状态, 甲方对接人, 电话, 邮箱。

## 4. 非功能性需求
- **响应式**: 适配桌面端浏览器（Chrome, Edge, Firefox）。
- **性能**: 页面加载时间 < 1秒（局域网环境）。
- **安全性**: 简单的基于 Token 的身份验证，防止未授权访问 API。
